    데이터 분석 에이전틱 수업¶

    데이터 분석 도구를 다루는 방법이 아니라 데이터로 생각하는 방법을 배우는 시간

    AI를 도구로, 학생을 주체로. 질문 설계부터 독립 프로젝트까지.

    학생들이 만들어낼 분석 결과 미리보기

    Before vs After: AI에게 어떻게 시키느냐가 결과를 바꾼다¶

    Before

    "AI야, 데이터 분석해줘"¶

    • 맥락 없이 던진 요청
    • 뒤죽박죽 라벨, 의미 불명의 차트
    • 결론도 근거도 없는 결과물
    • 학생은 구경꾼, AI가 주인

    After

    분석 지침서(Soul Document)로 구조화된 지시¶

    • 분석 목적, 변수, 기대 결과를 명시
    • 깔끔한 시각화 + 해석 포함
    • 근거 있는 결론 도출
    • 학생이 주인, AI는 도구
    같은 데이터, 다른 지시 — 결과 비교 체험

    4파트 학습 여정¶

    1

    질문을 설계하라 2.5시간¶

    상황카드로 문제를 발견하고, 3단 변환을 통해 막연한 궁금증을 분석 가능한 질문으로 완성합니다.

    2

    AI에게 분석을 시켜라 4시간¶

    분석 지침서(Soul Document)를 작성하여 AI에게 정확히 지시하고, 분석을 실행한 뒤, 결과를 스스로 검증합니다.

    3

    서로의 분석을 검증하라 2시간¶

    World Cafe 피어리뷰로 짝의 분석을 3가지 렌즈(데이터, 분석, 해석)로 비판적으로 검토합니다.

    4

    처음부터 끝까지 2시간¶

    독립 프로젝트로 질문 설계 → 데이터 수집 → AI 분석 → 해석 → 발표의 전체 파이프라인을 수행합니다.

    상황카드 6종 미리보기¶

    6가지 영역의 상황카드를 클릭하면 뒤집어집니다. 각 카드의 뒷면에는 분석 질문 예시가 있어요.

    상황카드 인터랙티브 체험

    핵심 체험 — 이상치 슬라이더¶

    슬라이더를 움직여 이상치를 제거하면 상관관계가 어떻게 변하는지 직접 확인하세요.

    이상치가 분석에 미치는 영향 — 직접 조작해 보세요

    이 수업의 특징¶

    🤖

    에이전틱 러닝¶

    학생이 AI에게 지시하는 주체입니다. AI는 도구일 뿐. "시켜봤더니 이상하다 → 왜 이상한가 → 어떻게 고칠까"를 반복하며 진짜 분석 역량을 키웁니다.

    📊

    데이터 리터러시¶

    숫자를 읽고, 시각화를 해석하고, 근거로 말하는 힘. 평균의 함정, 상관과 인과의 차이, 이상치의 영향까지 데이터를 비판적으로 바라보는 눈을 기릅니다.

    🏫

    실제 데이터¶

    학교 설문(200명), 서울 기온(11년), 수면과 성적 등 학생들이 공감할 수 있는 실제적 데이터로 수업합니다. 가짜 데이터가 아닌, 진짜 이야기가 담긴 숫자입니다.

    🔍

    비판적 검증¶

    피어리뷰를 통해 서로의 분석을 검증합니다. 데이터 무결성, 분석 논리, 해석 타당성의 3가지 렌즈로 비판적 사고력을 키웁니다.
