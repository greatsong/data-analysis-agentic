### 카드 상세

각 상황카드는 **앞면(상황 제시)**과 **뒷면(분석 질문 예시)** 으로 구성됩니다. 뒷면은 학생에게 바로 보여주지 않고, 활동 후 참고용으로 제공합니다.

#### 카드 1: 수면과 피로

**앞면 --- 상황**

> "요즘 1교시에 엎드려 자는 애들이 너무 많아. 다들 잠을 못 자나 봐."

- **영역**: 건강 / 생활습관
- **관련 데이터**: 수면시간, 학년, 피로도, 기상시간
- **난이도**: 하 (입문용으로 추천)

**뒷면 --- 질문 예시** (활동 후 공개)

- 1단계: 다들 잠이 부족한 것 같다
- 2단계: 학년이 올라갈수록 수면시간이 줄어드나?
- 3단계: 1학년과 3학년의 평일 평균 수면시간에 통계적 차이가 있는가?

- **관련 데이터**: 급식 만족도 점수, 메뉴 유형, 요일

**뒷면 --- 질문 예시**

- 1단계: 급식이 맛없는 것 같다
- 2단계: 요일에 따라 만족도가 다를까?
- 3단계: 요일별 급식 만족도 평균 점수에 유의미한 차이가 있는가?

- 3단계: 시험 기간의 일일 평균 스마트폰 사용시간이 평소보다 통계적으로 높은가?

#### 카드 4: 통학 스트레스

**앞면 --- 상황**

- **난이도**: 중

- 2단계: 통학시간이 길수록 스트레스가 높을까?
- 3단계: 통학시간과 스트레스 지수 사이에 양의 상관관계가 있는가?

#### 카드 5: 동아리 활동

**앞면 --- 상황**

> "동아리 활동을 하는 애들이 성적도 좋은 것 같은데, 진짜일까?"

- **영역**: 비교과 / 학업
- **관련 데이터**: 동아리 참여 여부, 주당 활동 시간, 학업 성취도
- **난이도**: 상

**뒷면 --- 질문 예시**

- 1단계: 동아리를 하면 성적이 좋아지나?
- 2단계: 동아리 활동 시간과 성적 사이에 관계가 있을까?
- 3단계: 주당 동아리 활동 시간과 평균 성적 사이에 상관관계가 있는가? (인과가 아닌 상관임을 유의)

!!! warning "교사 주의"
    이 카드는 **상관과 인과의 차이**를 가르치기 좋은 카드입니다. "동아리를 하면 성적이 좋아진다"는 인과적 주장이고, "동아리 활동과 성적 사이에 상관이 있다"는 관찰적 주장입니다. 학생들이 이 차이를 인식하도록 유도하세요.

- **영역**: 신체활동 / 집중력
- **관련 데이터**: 점심시간 활동 유형, 오후 집중도 자가평가, 학년

**뒷면 --- 질문 예시**

- 1단계: 점심에 운동하면 오후 수업이 더 잘 되는 것 같다
- 2단계: 점심시간 활동 유형에 따라 오후 집중도가 달라질까?
- 3단계: 점심시간 신체활동 그룹과 비활동 그룹의 오후 집중도 평균에 유의미한 차이가 있는가?

### 상황카드 활용 팁

!!! tip "모둠별 카드 배분 전략"
    - **전체 모둠 같은 카드**: 비교가 쉽고, 발표 시 차이점이 명확. 처음 수업에 추천.
    - **모둠별 다른 카드**: 다양한 질문이 나오고, 발표가 풍성. 수준이 높은 학급에 추천.
    - **모둠 내 개인 선택**: 자율성 최대. 시간이 넉넉한 경우에만 추천.

## 2. 3단 변환 활동지 (Part A)

### 활동지 양식

아래 양식을 인쇄하여 학생에게 배부합니다.

---

**[활동지 Part A] 3단 질문 변환**

이름: ________________  모둠: ____  상황카드 번호: ____

| 단계 | 내용 | 작성란 |
|------|------|--------|
| **1단계** | 막연한 느낌 | _________________________________ |
| | 상황카드를 읽고, 내가 느끼는 것을 한 문장으로 | |
| ↓ | *"이것을 어떻게 확인할 수 있을까?"* | |
| **2단계** | 구체적 궁금증 | _________________________________ |
| | 비교 대상이나 조건을 넣어 구체화 | |
| ↓ | *"이 궁금증에 답하려면 어떤 숫자가 필요할까?"* | |
| **3단계** | 데이터 질문 | _________________________________ |
| | 측정 가능한, 데이터로 답할 수 있는 질문 | |

**자가 체크**: 내 3단계 질문은...

- [ ] 무엇을 비교/측정하는지 분명하다 (명확성)
- [ ] 숫자로 표현할 수 있다 (측정가능성)
- [ ] 실제로 데이터를 모을 수 있다 (접근성)

---

### 좋은 예시 vs 보완 필요 예시

<div class="ba-before" markdown>

**상황카드: 수면과 피로**

2단계: (건너뜀)
3단계: 수면시간의 평균은?

**문제점**:

- 2단계를 건너뛰어 비교 대상이 없음

**상황카드: 수면과 피로**

2단계: 학년이 올라갈수록 더 적게 자는 걸까?
3단계: 1학년과 3학년의 평일 평균 수면시간에 통계적 차이가 있는가?

**강점**:

- 2단계에서 비교 대상(학년)을 특정
- 3단계에서 구체적 집단과 측정 변수를 명시
- 세 조건(명확성, 측정가능성, 접근성) 모두 충족

</div>

## 3. 데이터 역설계 활동지 (Part B)

### 활동지 양식

---

**[활동지 Part B] 데이터 역설계**

이름: ________________  모둠: ____

**내 데이터 질문** (Part A의 3단계 질문):

> _____________________________________________________________

| 항목 | 작성란 |
|------|--------|
| **필요한 변수** (무엇을 측정?) | 변수1: _____________ 변수2: _____________ |
| **데이터 유형** | [ ] 숫자(연속형)  [ ] 범주(그룹)  [ ] 예/아니오  [ ] 순위 |
| **조사 대상** (누구에게?) | _________________________________ |
| **수집 방법** (어떻게?) | [ ] 설문조사  [ ] 기존 기록 활용  [ ] 직접 관찰  [ ] 기타: _____ |
| **예상 표본 크기** (몇 명?) | ________명 (최소 ________명은 필요한 이유: _______________) |
| **예상 결과** (어떤 결과가 나올 것 같은가?) | _________________________________ |

**현실성 점검**:

- [ ] 이 데이터를 2주 안에 모을 수 있다
- [ ] 개인정보 침해 없이 모을 수 있다
- [ ] 우리 학교/반에서 충분한 표본을 확보할 수 있다

---

### 작성 예시

| 항목 | 예시 |
|------|------|
| **데이터 질문** | 학년별 평균 수면시간에 차이가 있는가? |
| **필요한 변수** | 변수1: 학년 (1/2/3학년) 변수2: 평일 평균 수면시간(시간) |
| **데이터 유형** | 학년: 범주(그룹), 수면시간: 숫자(연속형) |
| **조사 대상** | 우리 학교 1~3학년 학생 |
| **수집 방법** | 구글 폼 설문조사 |
| **예상 표본 크기** | 90명 (학년당 30명 이상 필요 --- 통계적 비교를 위해) |
| **예상 결과** | 3학년의 수면시간이 가장 짧고, 1학년이 가장 길 것이다 |

## 4. 질문 설계 루브릭

<div class="rubric-table" markdown>

| 평가 기준 | 탁월 (4) | 우수 (3) | 보통 (2) | 미흡 (1) |
|----------|---------|---------|---------|---------|
| **질문의 명확성** | 비교 대상, 측정 변수, 분석 방향이 모두 명시됨 | 비교 대상과 측정 변수가 명시됨 | 측정 변수만 언급, 비교 대상 불분명 | 질문이 모호하여 무엇을 분석해야 하는지 파악 불가 |
| **측정가능성** | 변수의 단위와 척도까지 구체적으로 설정 | 변수가 숫자로 측정 가능하도록 설정 | 일부 변수가 주관적이어서 측정 기준 필요 | 느낌/의견 수준에 머물러 숫자화 불가 |
| **데이터 접근성** | 수집 방법, 대상, 표본 크기를 현실적으로 설계 | 수집 방법과 대상은 명시, 표본 크기 미고려 | 수집이 가능하지만 구체적 계획 부족 | 현실적으로 데이터 수집이 어려운 질문 |
| **3단 변환 과정** | 3단계 전환이 논리적이고, 각 단계의 변화가 명확 | 3단계를 모두 거쳤으나 일부 전환이 약함 | 2단계를 건너뛰거나 1→3 직접 전환 | 1단계(느낌)에서 벗어나지 못함 |
| **데이터 역설계** | 변수, 유형, 대상, 방법, 표본, 예상결과 모두 구체적 | 대부분 항목 작성, 1~2개 항목 미흡 | 절반 정도 작성, 구체성 부족 | 역설계를 시도하지 못했거나 매우 불완전 |

</div>

!!! note "루브릭 사용 안내"
    - 이 루브릭은 **1차시 형성평가**용입니다.
    - 점수 합산이 아니라, **어떤 영역에서 추가 지원이 필요한지** 파악하는 데 사용합니다.
    - 특히 "질문의 명확성"과 "3단 변환 과정"이 미흡한 학생은 2차시에서 분석 지침서(Soul Document) 작성 시 추가 스캐폴딩(단계적 도움)이 필요합니다.

### 수준별 피드백 가이드

| 수준 | 종합 점수 | 교사 피드백 방향 |
|------|----------|----------------|
| **탁월** | 17~20점 | "훌륭해요. 2차시에서 이 질문 그대로 AI에게 시켜봅시다." |
| **우수** | 13~16점 | "좋은 질문이에요. 비교 대상이나 단위를 조금 더 구체화하면 완벽합니다." |
| **보통** | 9~12점 | "질문의 방향은 좋아요. 2단계(구체적 궁금증)를 다시 한번 다듬어 봅시다." |
| **미흡** | 5~8점 | "느낌을 잘 잡았어요. 이 느낌을 '어떤 숫자로 확인할 수 있을까' 함께 생각해 봅시다." |

## 5. 교사 커스터마이징 가이드

### 상황카드 교체/추가 방법

기본 제공 6종 외에 학교 상황에 맞는 카드를 추가할 수 있습니다.

!!! tip "좋은 상황카드의 조건"
    1. **학생이 공감하는 상황**: 실제로 학생들이 경험하거나 대화하는 주제
    2. **데이터로 바꿀 수 있는 상황**: 숫자, 범주, 빈도 등으로 측정 가능한 것
    3. **정답이 없는 상황**: "맞다/틀리다"가 아니라 "확인해 봐야 안다"는 느낌
    4. **적절한 복잡성**: 너무 단순하지도, 너무 복잡하지도 않은 것

### 상황카드 작성 템플릿

```
상황카드: [제목]

앞면 --- 상황:
"[학생의 말투로 쓴 한두 문장]"

- 영역: [건강/학교생활/디지털/교통/학업/기타]
- 관련 데이터: [측정 가능한 변수 3~4개]
- 난이도: [하/중/상]

뒷면 --- 질문 예시:
- 1단계: [막연한 느낌]
- 2단계: [구체적 궁금증]
- 3단계: [데이터 질문]
```

### 학교 맞춤 카드 예시

??? question "예시: 학교 축제 관련 카드"
    **앞면**: "작년 축제는 재미없었는데, 올해는 어떨까?"

    - 관련 데이터: 축제 만족도(5점), 선호 프로그램, 학년, 참여 시간
    - 난이도: 중

    **뒷면**:

    - 1단계: 축제가 재미없었다
    - 2단계: 학년에 따라 축제 만족도가 다를까?
    - 3단계: 학년별 축제 만족도 평균 점수에 유의미한 차이가 있는가?

??? question "예시: 체육 수업 관련 카드"

    - 영역: 학교생활 / 선호도
    - 관련 데이터: 선호 운동 종목, 실제 수업 종목 빈도, 성별, 만족도
    - 난이도: 하

    **뒷면**:

    - 1단계: 체육 시간에 축구만 한다
    - 2단계: 성별에 따라 선호하는 종목이 다를까?
    - 3단계: 남학생과 여학생의 선호 운동 종목 분포에 유의미한 차이가 있는가?
