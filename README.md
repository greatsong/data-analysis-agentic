# 데이터 분석 에이전틱 수업 — 교사용 지도서

AI를 도구로 활용하는 데이터 분석 4차시 수업. Plotly.js 인터랙티브 차트 15개, 4파트 10챕터로 구성되어 있습니다.

**사이트 바로가기**: https://greatsong.github.io/data-analysis-agentic/

## 개요

이 교재는 [에듀플로](https://github.com/greatsong/data-ai-book)로 제작된 교육 자료입니다.
AI를 도구로 활용하는 데이터 분석 4차시 수업. Plotly.js 인터랙티브 차트 15개, 4파트 10챕터로 구성되어 있습니다.
모든 콘텐츠는 오픈소스로 공개되어 있으며, 누구나 자유롭게 활용하고 개선에 참여할 수 있습니다.
